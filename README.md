# Flask Login & Comments App

## Overview
This is a beginner Flask web application that demonstrates basic user login functionality and a simple comment viewing system.

## Features
- Login system with predefined users
- Display user-specific comments
- Simple Flask routing and templates

## Tech Stack
- Python (Flask)
- HTML (Jinja templates)
- PythonAnywhere (deployment)

## Live Demo
https://faridkasdi.pythonanywhere.com/

## Project Purpose
This project was built as part of my learning journey in cloud and DevOps training to understand how Flask web applications work and how to deploy them online.
